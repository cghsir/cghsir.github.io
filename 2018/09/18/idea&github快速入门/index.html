git在idea中快速入门，并连接github

18 Sep 2018
  • 前提已经在windows电脑安装git应用，并且在文件夹右键有git

  • 设置git的全局名称与邮箱，邮箱不必存在并不会验证。任意目录右键-》Git Bash Here。然后输入如下命令设置。

  • $ git config –global user.name “git_test”

  • $ git config –global user.email “git@qq.com“

  • 打开github官网，并注册账号。官网https://github.com/ 。注册过程略

  • 登录之后开始一个项目

  • 创建一个新的仓库

  • 创建成功后

  • 远程仓库已经建好接下来将本地的git与github进行关联操作。配置ssh

  • 将如下红框的内容复制备用

  • 进行测试

  • 找到自己的仓库然后点这个按钮复制URL

    在计算机上找个文件夹。执行命令 git clone +刚刚那个复制的url

    下载成功

  • 打开idea配置git

  • 然后将刚刚下载的目录作为项目用idea打开

File -> Open

然后新建一个index.html 然后选择是

然后再版本控制选择新增的绿色的index.html。右键commit

绿色代表新增的文件。蓝色代表变动的文件。灰色代表删除的文件

填写提交备注然后提交，注意这里的提交只是提交的本地仓库并没有提交到远程仓库。

将这个文件推送到github上，在推送之前先看下log

这个log中 黄色的代表本地所在的位置，orgin代表服务器所在的记录位置。将本地变更推送到服务器上。

项目右键 -> Git -> Repository -> Push

push完后log的变化
